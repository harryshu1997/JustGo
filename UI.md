## 6. Apple Watch · 进行中（buddy 陪伴模式）

**核心交互（三屏共用）**：

- **单点屏幕** → 暂停（buddy 进入"等待"姿势，计时停下，再单点恢复）
- **长按 1 秒** → 触觉 `.retry` + 弹出"确认结束?" 全屏二选项（结束 / 取消）
- **Digital Crown 滚** → 次数型 +N，时长型快速浏览预计剩余时间

> 设计原则：单点低成本（暂停可恢复），长按高成本（结束不可恢复），用物理输入差异防误触，不用图形按钮挤占小屏。

### 7.3 完成后台流程链路

1. 长按"结束"确认 → Watch 本地立即写草稿 `WorkoutSession`（即使 iPhone 离线也立刻反馈）
2. 播完成动画 + buddy 抱树苗跑出（视觉/听觉立即满足，不等同步）
3. `transferUserInfo` 推送给 iPhone（系统队列保证最终送达）
4. iPhone 收到 → SwiftData 写入 → 写日历 → 计算积分 + buddy 经验 → 种树 → 推送下一目标
5. Watch 收到下一目标 → 显示"下一个目标"提示屏

**今日队列最后一个完成时**：跳过下一目标提示，直接进入"今日已完成"庆祝屏（buddy 大跳 + 解锁今日特别奖励，比如稀有树种）。

### 9.3 实现骨架（可直接粘到 Xcode Preview）

```swift
import SwiftUI

enum PixelColor: Int {
    case empty = 0, trunk, primaryDark, primary, primaryLight,
         sakuraLight, sakuraDark
}

struct PixelTree: View {
    let pattern: [[PixelColor]]
    var pixelSize: CGFloat = 6

    var body: some View {
        VStack(spacing: 0) {
            ForEach(0..<pattern.count, id: \.self) { y in
                HStack(spacing: 0) {
                    ForEach(0..<pattern[y].count, id: \.self) { x in
                        Rectangle()
                            .fill(color(for: pattern[y][x]))
                            .frame(width: pixelSize, height: pixelSize)
                    }
                }
            }
        }
    }

    private func color(for c: PixelColor) -> Color {
        switch c {
        case .empty:        return .clear
        case .trunk:        return Color(red: 0.42, green: 0.26, blue: 0.15)
        case .primaryDark:  return Color(red: 0.14, green: 0.54, blue: 0.24)
        case .primary:      return Color(red: 0.20, green: 0.78, blue: 0.35)  // #34C759
        case .primaryLight: return Color(red: 0.42, green: 0.88, blue: 0.55)
        case .sakuraLight:  return Color(red: 1.00, green: 0.75, blue: 0.80)
        case .sakuraDark:   return Color(red: 1.00, green: 0.48, blue: 0.66)
        }
    }
}

extension PixelTree {
    static let oakStage3: [[PixelColor]] = {
        let e = PixelColor.empty
        let D = PixelColor.primaryDark
        let G = PixelColor.primary
        let L = PixelColor.primaryLight
        let B = PixelColor.trunk
        return [
            [e,e,e,D,G,G,L,e,e,e],
            [e,e,D,G,G,G,L,L,e,e],

### 9.4 像素动画思路

- **生长**：从底向上逐行揭开（`.mask` + 时间驱动的高度）
- **风吹**：每 3 秒整树轻微 ±1 像素左右偏移（`offset` 动画 + spring）
- **种树时**：种子 → 嫩芽阶段用 `.scale` + `.opacity` 弹入
- **摇曳**：完成 session 弹"完成！"时全树触发 5° 倾斜回弹

### 9.5 像素风对 SF Symbol UI 的影响

像素树是**森林视图独家**风格化资产，App 其他地方（按钮、tab、图标）依然用 Apple HIG 推荐的 SF Symbols + 现代字体，避免整个 App 显得太复古。这种"内容像素化、UI 现代化"的混搭模式 Pokémon Sleep / 苹果自家 Playgrounds 都用过，效果稳。
